# Grok — use as custom instructions

Grok supports **Customize Grok** or **Workspace** with custom instructions. Paste the content of **Custom_Instructions.md** into "How would you like Grok to respond?".

## Install (recommended)

1. Open **Customize Grok** (or create a **Workspace**).
2. In "How would you like Grok to respond?", paste the content of **Custom_Instructions.md**.
3. Save. Grok will follow the Nano Banana image guide workflow.

## How to use

- Say e.g. "I want to create a product mockup" or "Generate an image of…"
- Grok will ask questions one at a time, then build the prompt (and generate if Grok has image generation).

## If you don't use custom instructions

- In the first message, paste the one-shot prompt from the main **README.md** and add your request.
- Or paste **Image_Guide_Playbook.md** if the interface allows.
