# Claude (claude.ai) — use as Skill

## Install (recommended)

1. **Settings > Capabilities:** Enable *Code execution and file creation*.
2. **Customize > Skills:** Click *Add skill* (or similar). Upload this folder as a **ZIP** (must include `SKILL.md` and `Image_Guide_Playbook.md`).
3. Done. Claude will use the skill automatically when you ask to create or edit an image. No need to invoke it explicitly.

## Alternative: Project or attach

If you don't use Skills: create a **Project**, paste the instructions from **Project_Instructions.md** into Project instructions, add **Image_Guide_Playbook.md** to Knowledge. Or simply attach **Image_Guide_Playbook.md** in any chat and ask Claude to follow it.
