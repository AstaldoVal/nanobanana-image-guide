# ClawdBot — workaround (skills are code-based)

ClawdBot skills use manifest.json + code, not a single markdown skill. Use **Custom instructions** if ClawdBot supports them: paste the content of **Custom_Instructions.md**. Otherwise attach **Image_Guide_Playbook.md** in the chat or use the one-shot prompt from the main **README.md**.

## Install

1. If ClawdBot has custom instructions: paste **Custom_Instructions.md**.
2. Otherwise: in each relevant chat, attach **Image_Guide_Playbook.md** and say "Use this playbook to help me create an image…"
