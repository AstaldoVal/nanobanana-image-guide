# Manus — use as a skill

Manus supports **Skills** (upload a folder with SKILL.md). The Nano Banana Image Guide is ready to upload.

## Install (recommended)

1. In Manus → **Skills** → **Upload**.
2. Upload the **standalone folder** (the one that contains `SKILL.md` and `Image_Guide_Playbook.md`), or a .zip of it.
3. The skill appears in chat (e.g. `/nanobanana-image-guide`).

## How to use

- Invoke the skill (e.g. `/nanobanana-image-guide` or "use Nano Banana image guide").
- The assistant will follow the playbook: determine task type → ask questions one at a time → build prompt → generate or output prompt.

## If upload isn't available

- Paste the contents of **Image_Guide_Playbook.md** in the chat, or use the one-shot prompt from the main **README.md**.
