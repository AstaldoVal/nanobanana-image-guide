# Gemini — use as a Custom Gem (skill)

Gemini supports **Custom Gems** with instructions. The Nano Banana Image Guide is ready to paste into a Gem.

## Install (recommended)

1. In Gemini, create a **Gem** (or edit an existing one).
2. In the Gem's **Instructions** (or "How should this Gem respond?"), paste the content of **Custom_Instructions.md**.
3. Save. Use this Gem whenever you want to create or edit images with structured prompting.

## How to use

- Open the Gem and say e.g. "I want to create a product mockup" or "Generate an image of…"
- The assistant will ask questions one at a time, then build the prompt (and generate if Gemini has image generation).

## If you don't use Gems

- Paste **Image_Guide_Playbook.md** in the chat and ask Gemini to follow it.
- Or use the one-shot prompt from the main **README.md**.
